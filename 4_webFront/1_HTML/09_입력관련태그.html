form태그

            -div 와 같은 영역 (block형식)
            + 내부에 작성된 input 태그의 값을 서버로 전달하는 역할 

            [속성]
            1) action: 내부에 작성된 값을 전달할 서버또는 페이지 주소를 작성
            2) method(방식) : 서버로 데이터 전달하는 방식을 지정하는 속성
            3) name :form태그에 이름을 지정하여 각각의 form태그 구분(JS사용)
            4) taget : action에 지정된 두소를 새탭/ 현재탭 중 어디서 열지 지정
