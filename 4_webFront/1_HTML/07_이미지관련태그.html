img 태그


        -웹문서에 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

        [속성]
        1) src :삽입할 이미지의 경로를 지정하는 속성(파일경로, 웹 주소)
        2) width/ height :이미지의 크기를 지정하는 속성
        3) alt :이미지를 설명하는 속성
                -> 이미지가 출력 되지 않는 경우 화면에 직접 표시
                    (+웹 전근성)

src 속성 확인

width, height을 이용한 크기 조정(고정, 가변 크기)

고정크기 단위(px) :화면의 크기 상관 없이 지정된 크기를 갖는 단위

가변크기 단위(%) : 화면이나 영역기준 사이즈의 크기에 따라서 크기가 변경되는 단위

alt 속성 확인
