영역을 나누는 형식(성질)

        
block 형식

        - 공간을 수직 분활
        -> 수직으로 되어있는 화면을 가로로 잘라서 
           여러행을 만드는것
        - 크기를 지정하는 width, heigth 속성을 사용할 수 있음(O)

        ex) p/pre (문단 나누기), h1~h6(제목 나누기),
        hr(수평선을 넣어서 화면나누기), div(영역나누기)

    
        
inline 형식

        - 공간을 수평 분할 
        -> 수평으로 되어있는 화면을 새세로로 잘라서 
           여러 열(컬럼)을 만드는것

        - 크기를 지정하는 width,height 속성을 사용할 수 없음(x)

        ex) strong /b(글자를 굵게), em/i(글자 기울기),
            mark(글자에 배경색),span(한줄을 나누는 용도)
