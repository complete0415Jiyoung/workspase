ol:순서가 있는 목록을 작성하는 태그(영역)

            기본값 : 1부터 1씩 증가하는 숫자 
            "ol" 태그의 type 속성

            type="a" -->영어 소문자 순서
            type="A" --> 영어 대문자 순서
            type="i" --> 영어 대문자 순서


        +추가 속성
        start ="5"--> 순서가 5 부터 시작
        reversed --> 순서가 반대로
