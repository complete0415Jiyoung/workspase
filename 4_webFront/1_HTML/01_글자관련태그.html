글자 관련 기타 태그

태그 : 단순히 글자를 굵게 표시하는 태그
strong 태그 :글자를 장조하는 태그(웹접근성)
i태그 :단순히 글자를 기울이는 태그
em태그: 글자를 강조하는 태그 (웹접근성)
mark태그 :형광팬 효과를 나타내는 태그
u 태그 :밑줄을 긋는 태그
s태그 :취소선을 긋는 태그 (비슷한 태그 del 존재)
del 태그: small태그:글자를 작게
sup 태그 (윗첨자): 2
sub 태그 (아래첨자): log10
abbr(abbreviation:막어) 태그 : 마우스 오버시 올림 출력 태그 HTTP
