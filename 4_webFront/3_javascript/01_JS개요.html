JS개요

스크립트(script) 언어

        기본프로그램의 동작을 사용자의 요구에 맞게 특정 기능을 수행 되도록 해주는 용도의 언어

        별도의 소스 코드를 컴파일 하지 않고 
        인터프리터(interpreter)을 이용하여 소스 코드를 한줄씩 읽어 실행
        ->  컴파일에 소요 되는 시간이 없음
            대신 프로그램 실행 시간이 느림
        
        짧은 소스코드 파일로 상호작용 하도록 보완
        
        인터프리터(interpreter) :코드를 해석하는 친구

자바스크립스란(JS)란?

        웹 브라우저에서 많이 사용되는 인터프리터 방식의 객체지향 프로그래밍 언어
        == 웹브라우저에서 많이 사용되는 인터프리터 방식의 스크립트 언어
        - 인터프리터 방식: 코드를 한줄 한줄 읽어 가며 바로 실행하는 방식
    
        ECMA Script 표준을 따르는 대표적인 웹 기술이다

자바스크립스 작성방법

  • 브라우저 콘솔에 직접 작성하는 방법
  • html 내부에서 script 태그를 이용하여 작성(internal)
  • html 외부에(.js)파일을 이용해서 작성(external)
  • 태그에 직접 JS코드를 작성(inline)

1) inline방식

        태그내에 간단한 소스 코드를 작성해서 실행되게 하는 방법
        [표현법] < 태그 on 이벤트명 = "해당 요소에 해당 이벤트 발생시 실행할 소스 코드 "⁢

2) internal 방식

        현재 html문서 내에 script 태그 영역 안에 소스코드 작성
        script 태그 head 태그 body 태그든 다 작성 가능

3) external방식

        별도의 .js파일로 소스코드 를 작성해서 가져다가 사용하는 방법
