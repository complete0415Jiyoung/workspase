자바스크립트 객체

        자바스크립트의 객체는 {} 내에 Key:Value 가 모여있는 형태로 작성된다.
                                    (Map 형식)

        { K:V , K:V , K:V } // 자바스크립트 객체

        (참고) 자바스크립트 객체 모양의 문자열
        == JSON(JavaScript Object Notation, 자바스크립트 객체 표기법)
        -> "{ K:V , K:V , K:V }"


        - 자바스크립트에 객체를 생성하는 방법

        1. {} : 객체 리터럴 표기법을 이용한 생성
        2. 생성자 + new 생성자()를 이용한 생성
