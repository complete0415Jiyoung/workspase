배열(Array)

        배열 변수을 묶음으로 다루는 것 (변수가 연속적으로 나열됨)

        *JS 변수 : 값이 지정되기 전까지 자료형이 지정되지 않음

        JS 배열 특징
        1. 자료형 제한X
        2. 길이 제함 X

배열 관련 함수

        (Stack구조 관련 함수)
        push() : 배열 마지막 요소로 추가
        pop() : 배열의 마지막 요소를 꺼내옴

        배열.indexOf("값") : 일치하는 값을 가진 요소의 index 를 반환
        없으면 -1 반환


        배열.sort([정렬기준 함수]) : 배열 내의 요소를 오름차순 정렬(문자열)
                            단, [정렬기준함수]가 작성되면 정렬 결과가 달라짐

        배열.toString() : 배열 요소를 하나의 문자열로 출력
                            ->요소 사이에 ","" 추가 
        배열.join("구분자") : 배열 요소를 하나의 문자열로 출력
                            -> 요소 사이에 "구분자" 추가
