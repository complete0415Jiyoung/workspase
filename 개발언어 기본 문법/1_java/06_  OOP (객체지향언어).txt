객체(Object) : 독립적으로 각각 구분해서 인식할 수 있는 모든 것
// - 물건 , 물체, 요소, 대상 , 목적 
ex) 키보드, 김치, 나(자신), 조원, 미세먼지, 모래 한알, 판소리 무형문화제


지향(Oriented) : 어떠한 것을 목표로 함

객체지향 :  독립적으로 각각 구분해서 인식할 수 있는 것을 목표로함

(컴퓨터) 언어 (Language) : 코드, 코딩 방식



*******
객체 지향 언어 : 독립적으로 각각 구분해서 인식할 수 있는 것을 목표로 하는 코딩 방식
	현실 세계는 사물이나 개념처럼 독립되고 구분되는 각각의 객체로 이루어져 있으며,
	발생하는 모든 사건들은 객체간의 상호작용이다.
	이 개념을 컴퓨터로 옮겨 놓아 만들어낸 것이 객체지향 언어이다


객체 지향의 3+1대 특징 
1) 캡슐화 
2) 상속
3) 다향성
4) 추상화

현실에서 객체란?
자신의 속성(값, data)과 기능(동작,행동)이 다른 것들 과 구분되어 식별 가능한 

ex) 나
(속성) 이름 , 나이 ,생년월일   	
(기능) 밥을 먹는다. 강의 듣는다, 잠 잔다, 맥주 마신다 

모니터 
(속성) 크기, 무게, 화소		
(기능) 화면을 보여준다. 밝기를 조절한다. 

자바에서 객체란?
클래스에 정의된 내용대로 new 연산자를 통해 heep 메모리 영역에 생성된 것
->new연산자를 사용한 모든 것은 객체
	ex) Arr2Practice arr2 = new  Arr2Practice(); -> arr2 이 객체
	ex) int[] arr = new int[4]; -> arr객체


클래스 ( == 설계도 ) : 객체의 특성(속성, 기능)에 대한 정의를 한 것
클래스 생성 시 반드시 필요한 것 - 추상화(abstraction), 캡슐화(encapsulation)이 적용
	-추상화: 프로그램에서 필요한 부분을 추출하고 불필요한 부분 제거 (유연성)
